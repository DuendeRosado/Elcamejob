  El CameJob

El CameJob es una plataforma digital diseñada para conectar personas que necesitan servicios con trabajadores independientes, facilitando la contratación de manera ágil, segura y eficiente.

##  Propósito

Este proyecto nace como una solución tecnológica frente al desempleo y la informalidad laboral en Colombia, permitiendo a los trabajadores ofrecer sus servicios y a los clientes encontrar soluciones confiables.

---

## ⚙️ Tecnologías utilizadas

### Backend
- Node.js
- Express
- MySQL
- JSON Web Tokens (JWT)
- Bcrypt

### Frontend
- HTML / JavaScript (base)
- Escalable a React

---

## Funcionalidades principales

- ✅ Registro de usuarios (cliente / trabajador)
- ✅ Inicio de sesión con autenticación JWT
- ✅ Publicación de servicios
- ✅ Visualización de servicios disponibles
- ✅ Contratación de trabajadores
- ✅ Seguridad básica implementada

---

## Arquitectura del sistema

El proyecto sigue una arquitectura tipo:

Cliente → API REST → Base de datos

Organizado en:
- Controllers (lógica)
- Routes (endpoints)
- Middleware (seguridad)
- Config (base de datos)

---

## Base de datos

Incluye:
- Usuarios
- Servicios
- Contrataciones
